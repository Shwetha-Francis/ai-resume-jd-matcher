# AI Resume – Job Description Matcher

This project uses NLP and machine learning to compare resumes with job descriptions and calculate a match score.

Features
- Resume parsing
- Text preprocessing using spaCy & NLTK
- TF-IDF similarity
- Sentence Transformers embeddings
- Skill gap detection
- Streamlit web app

Tech Stack
Python
spaCy
NLTK
Scikit-learn
Sentence Transformers
Streamlit

Run the app

streamlit run resume_app.py
